# log4cplusplus

基于 log4cplus 封装的自用的 C++ 日志库

## Feature

* [x] 线程安全
* [x] 支持异步
* [x] 支持中文路径和内容
* [x] 支持输出到文件、控制台、调试器
* [x] 支持格式化打印

## Clone

```
git clone --recursive https://github.com/huihut/log4cplusplus.git
```

## Build

```
./generate_sln.bat
```
